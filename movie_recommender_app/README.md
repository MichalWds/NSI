# HOW TO RUN 
 From main directory use 
 `python3 main.py --user USER --similarity-type {euclidean,pearson,Euclidean,Pearson,EUCLIDEAN,PEARSON}` 
  command. 

  _example:_ 
 `python3 main.py -u "Michał Wadas" -sc 'Pearson'`
